# rostam-assets

Character and environment art for **The Seven Labors of Rostam** (Haft Khan) —
a narrative mobile game based on Ferdowsi's *Shahnameh*.

**Art style:** Safavid Tabriz miniature (c. 1530).

## Character assets

| File | Character | Status |
|------|-----------|--------|
| `rostam.png` | Rostam | Locked |
| `arzhang.png` | Arzhang Div | In review |
| `div_white.png` | Div-e Sefid (White Div) | In review — approved take avoids European grotesque styling |
| _pending_ | Rakhsh | Locked (not yet uploaded) |

## Usage

Reference an asset by its raw URL:

`https://raw.githubusercontent.com/aryatechnologiesllc-arch/rostam-assets/main/<filename>`

Example: `.../main/rostam.png`# rostam-assets
